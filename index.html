Open Science Workshops

Welcome

We are launching Open Science Workshops in order to bring open source techniques to the science community. We want to show that it is possible to do open science now, and help everyone explore all the really excellent tools out there that aid the research process and promote openness.

Goal

Contribute to the betterment of the world by making it easier for people to do science; i.e. make it easy to replicate results, understand results, collaborate, and build upon existing work.

Check out the longer term goals if you want more detail.

The event

It's completely free, and being held on the 19th of July, 2014, in Melbourne, Australia, at Inspire9 (Level 1, 41 Stewart Street, Richmond).

For those who can't attend, the hashtag will be #OpenScienceMelb.

Here's a video of Noon Silk talking about the event!

Open Science Workshop - Melbourne, 19th of July on Vimeo.

Why should you attend?

You should attend because the open source world is freely providing many amazing tools that can directly help with the research process! In particular, they can help with the organisation of work, efficiency of collaboration, and general enjoyment of life!

Agenda

The day is going to be focused on understanding, and getting started with, two essential concepts:

  • collaboration on documents, code and data
  • running and working with code and data in a collaborative environment

To do this, we will be using GitHub and the SageMathCloud. We will present and work with these concepts assuming no background knowledge.

Full breakdown of the day

You will need to bring a laptop (and preferably be able to install things on it.)

Help with the workshop!

We are currently looking for volunteers to be involved in all aspects of this project! For the upcoming workshop in Melbourne, if you are willing to:

  • talk about GitHub and help with setup,
  • talk about IPython, help with basics,
  • be excited and generally help out,
then we want to hear from you! To get involved, email Noon Silk, or even better just head over to the project on GitHub and start forking and pull requesting things!

Who is behind this?

Check out the about page for more information on this! The bottom line is - we all are! Let's all work together to get wider adoption of open source software and attitudes in the sciences and in society at large!
